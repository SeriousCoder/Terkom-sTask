Проект состоит из GUI (WPF) и Dll, реализованной на Cuda.

Программа воспроизводит видео, берет кадр и нормализует видео относительно гистограммы взятого кадра.

На момент написания данного ReadMe:

Реализовано:
- Загрузка, возпроизвидение видео (правда фильтр настроен только под mp4 файлы)
- Захват кадра и отображение его. (в GUI)
- Построение карты перекодировки.

Проблемы на данный момент:
- Не выходит взять фрейм из видео потока. Получается взять начальный кадр (TimeSpan = 0:00:00), остальные не берет (с другими значениями TimeSpan). 
Из-за этого не получается выделить все фреймы, чтобы применить нормализацию к видео...  (Для работы с видео потоком взят "MediaToolkit")


В качестве примера в папке Resources лежит видео (это было единственное видео на моем компьюторе)
